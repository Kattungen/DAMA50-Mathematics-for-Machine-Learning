# DAMA50-Mathematics-for-Machine-Learning


This repository contains my coursework and practical assignments for the course **Mathematics for the Maching Learning**, part of the MSc program **Data Science and Machine Learning** at Hellenic Open University (HOU), academic year 2025-2026. 

The course focuses on the mathematical foundations of machine learning, including linear algebra, calculus, probability, statistics, and optimization, with all assignments implemented in SageMath using Jupyter Notebook. The goal is to strenghten mathematical intuition and bridge theory with computational implementation.


## References
- Deisenroth M.P., Faisal A.A., Ong C.S. *Mathematics for Machine Learning*. Cambridge: Cambridge University Press, 2020.
- Zimmermann P., Casamayou A., Cohen N., et al. *Computational Mathematics with SageMath*. Philadelphia: SIAM, 2018.

## Additional Resources
- SageMath Official Documentation: [https://doc.sagemath.org/](https://doc.sagemath.org/)
- YouTube tutorials
- Personal lecture notes and course slides

## Topics covered
- **Lineal Algebra**: vectors, matrices, eigenvalues, decompositions
- **Calculus**: derivatives, partial derivatives, gradients
- **Probability and Statistics**: basic probability, distributions, expected value, variance
